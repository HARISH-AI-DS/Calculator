
Calculator Program – README
Overview
This Calculator Program is a simple tool that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It is created to help beginners understand programming logic and mathematical operations in a clear and easy manner.

Features
Addition
Subtraction
Multiplication
Division

User-friendly interface

Clean and modular code structure

Technologies Used

Programming Language: Python

Concepts used: Functions, Input/Output, Error handling, Mathematical operations

Project Structure
Calculator/
│── calculator.py
│── README.md
└── assets/ (optional)

How to Run
For Python:
python calculator.py

How It Works
The user selects an operation.
The user enters two numbers.
The program performs the selected operation using predefined functions.
The result is displayed.
The program repeats until the user decides to exit.
Error Handling
Handles invalid inputs
Prevents division by zero
Displays proper error messages
